# Pedestrian-Risk-Score---Civic-Data-Analysis
Final project from PPUA 5262 - Big Data for Cities


Used RStudio packages (tidyverse, ggplot2, sf, leaflet, lm) to analyze Boston 911 call records and develop a community-informed metric: the pedestrian risk score. This report summarizes the creation and utility of this metric, and its potential uses in policy and planning.
